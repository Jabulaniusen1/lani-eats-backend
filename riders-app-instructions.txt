# Lanieats Rider App — Windsurf Build Prompt

---

## PROJECT OVERVIEW

Build the **Lanieats Rider App** — a React Native mobile application for delivery riders on the Lanieats food delivery platform in Nigeria. Riders use this app to go online, see available deliveries, accept them, navigate to restaurants and customers, and mark deliveries complete.

This is a **production-ready** app. Build everything fully functional, fully connected to the API, with a polished UI optimized for one-handed use while on a motorbike.

## API SETUP

### Base URL
```
http://localhost:8000/api/v1
```

### `src/api/client.js`
```js
import axios from 'axios';
import { getToken, clearAuth } from '../utils/storage';

const BASE_URL = 'http://localhost:8000/api/v1';

const client = axios.create({
  baseURL: BASE_URL,
  timeout: 15000,
  headers: { 'Content-Type': 'application/json' },
});

client.interceptors.request.use(async (config) => {
  const token = await getToken();
  if (token) config.headers.Authorization = `Bearer ${token}`;
  return config;
});

client.interceptors.response.use(
  (res) => res,
  async (error) => {
    if (error.response?.status === 401) {
      await clearAuth();
    }
    return Promise.reject(error);
  }
);

export default client;
```

### `src/utils/storage.js`
```js
import AsyncStorage from '@react-native-async-storage/async-storage';

export const saveAuth = async (token, user) => {
  await AsyncStorage.multiSet([
    ['token', token],
    ['user', JSON.stringify(user)],
  ]);
};

export const getToken = () => AsyncStorage.getItem('token');

export const getUser = async () => {
  const user = await AsyncStorage.getItem('user');
  return user ? JSON.parse(user) : null;
};

export const clearAuth = async () => {
  await AsyncStorage.multiRemove(['token', 'user']);
};
```

## SOCKET.IO SETUP

### `src/services/socket.js`
```js
import { io } from 'socket.io-client';

const SOCKET_URL = 'http://localhost:8000';
let socket = null;

export const connectSocket = () => {
  if (!socket) {
    socket = io(SOCKET_URL, {
      transports: ['websocket'],
      reconnection: true,
      reconnectionAttempts: 10,
      reconnectionDelay: 2000,
    });

    socket.on('connect', () => console.log('🔌 Rider socket connected'));
    socket.on('disconnect', () => console.log('❌ Rider socket disconnected'));
  }
  return socket;
};

export const getSocket = () => socket;

export const joinRiderRoom = (riderId) => {
  const s = getSocket();
  if (s) s.emit('join_rider_room', riderId);
};

export const disconnectSocket = () => {
  if (socket) {
    socket.disconnect();
    socket = null;
  }
};
```

## LOCATION SERVICE

### `src/services/location.js`
```js
import * as Location from 'expo-location';
import client from '../api/client';

let locationSubscription = null;

export const requestLocationPermission = async () => {
  const { status } = await Location.requestForegroundPermissionsAsync();
  return status === 'granted';
};

export const getCurrentLocation = async () => {
  const location = await Location.getCurrentPositionAsync({
    accuracy: Location.Accuracy.High,
  });
  return {
    latitude: location.coords.latitude,
    longitude: location.coords.longitude,
  };
};

// Call this when rider starts an active delivery
// Sends location to backend every 10 seconds
export const startLocationTracking = async () => {
  stopLocationTracking();

  locationSubscription = await Location.watchPositionAsync(
    {
      accuracy: Location.Accuracy.High,
      timeInterval: 10000,     // every 10 seconds
      distanceInterval: 50,    // or every 50 meters
    },
    async (location) => {
      try {
        await client.patch('/rider/location', {
          latitude: location.coords.latitude,
          longitude: location.coords.longitude,
        });
      } catch (e) {
        console.log('Location update failed', e.message);
      }
    }
  );
};

export const stopLocationTracking = () => {
  if (locationSubscription) {
    locationSubscription.remove();
    locationSubscription = null;
  }
};
```

## ZUSTAND STORES

### `src/store/auth.store.js`
```js
import { create } from 'zustand';
import { saveAuth, clearAuth, getToken, getUser } from '../utils/storage';
import { loginApi, registerApi, getMeApi } from '../api/auth.api';
import { connectSocket, joinRiderRoom, disconnectSocket } from '../services/socket';

const useAuthStore = create((set) => ({
  user: null,
  rider: null,
  isAuthenticated: false,
  isLoading: true,

  initialize: async () => {
    try {
      const token = await getToken();
      if (token) {
        const res = await getMeApi();
        const { user } = res.data.data;
        await saveAuth(token, user);
        set({ user, rider: user.rider, isAuthenticated: true });
        connectSocket();
        if (user.rider) joinRiderRoom(user.rider.id);
      }
    } catch {
      await clearAuth();
    } finally {
      set({ isLoading: false });
    }
  },

  login: async (email, password) => {
    const res = await loginApi(email, password);
    const { token, user } = res.data.data;
    await saveAuth(token, user);
    // Fetch full profile to get rider object
    const meRes = await getMeApi();
    const fullUser = meRes.data.data.user;
    set({ user: fullUser, rider: fullUser.rider, isAuthenticated: true });
    connectSocket();
    if (fullUser.rider) joinRiderRoom(fullUser.rider.id);
  },

  register: async (data) => {
    const res = await registerApi({ ...data, role: 'RIDER' });
    const { token, user } = res.data.data;
    await saveAuth(token, user);
    const meRes = await getMeApi();
    const fullUser = meRes.data.data.user;
    set({ user: fullUser, rider: fullUser.rider, isAuthenticated: true });
    connectSocket();
    if (fullUser.rider) joinRiderRoom(fullUser.rider.id);
  },

  logout: async () => {
    await clearAuth();
    disconnectSocket();
    set({ user: null, rider: null, isAuthenticated: false });
  },

  refreshRider: async () => {
    const res = await getMeApi();
    const fullUser = res.data.data.user;
    set({ user: fullUser, rider: fullUser.rider });
  },
}));

export default useAuthStore;
```

### `src/store/delivery.store.js`
```js
import { create } from 'zustand';
import {
  getAvailableDeliveriesApi,
  acceptDeliveryApi,
  updateDeliveryStatusApi,
  getMyDeliveriesApi,
} from '../api/delivery.api';

const useDeliveryStore = create((set, get) => ({
  availableDeliveries: [],
  activeDelivery: null,
  deliveryHistory: [],
  isLoading: false,

  fetchAvailable: async () => {
    set({ isLoading: true });
    try {
      const res = await getAvailableDeliveriesApi();
      set({ availableDeliveries: res.data.data.deliveries });
    } finally {
      set({ isLoading: false });
    }
  },

  fetchHistory: async () => {
    set({ isLoading: true });
    try {
      const res = await getMyDeliveriesApi();
      set({ deliveryHistory: res.data.data.deliveries });
    } finally {
      set({ isLoading: false });
    }
  },

  acceptDelivery: async (deliveryId) => {
    const res = await acceptDeliveryApi(deliveryId);
    const delivery = res.data.data.delivery;
    set({ activeDelivery: delivery });
    return delivery;
  },

  updateStatus: async (deliveryId, status, coords) => {
    const res = await updateDeliveryStatusApi(deliveryId, status, coords);
    const delivery = res.data.data.delivery;
    if (status === 'DELIVERED' || status === 'FAILED') {
      set({ activeDelivery: null });
    } else {
      set({ activeDelivery: delivery });
    }
    return delivery;
  },

  setActiveDelivery: (delivery) => set({ activeDelivery: delivery }),

  addAvailableDelivery: (delivery) => {
    set((state) => ({
      availableDeliveries: [delivery, ...state.availableDeliveries],
    }));
  },

  removeAvailableDelivery: (orderId) => {
    set((state) => ({
      availableDeliveries: state.availableDeliveries.filter(
        (d) => d.order.id !== orderId
      ),
    }));
  },
}));

export default useDeliveryStore;
```

## UTILITY FUNCTIONS

```js
export const formatCurrency = (amount) =>
  `₦${Number(amount).toLocaleString('en-NG')}`;

export const formatOrderNumber = (id) =>
  `#${id.slice(-4).toUpperCase()}`;

export const formatRelativeTime = (date) => {
  const seconds = Math.floor((new Date() - new Date(date)) / 1000);
  if (seconds < 60) return 'Just now';
  if (seconds < 3600) return `${Math.floor(seconds / 60)}m ago`;
  if (seconds < 86400) return `${Math.floor(seconds / 3600)}h ago`;
  return new Date(date).toLocaleDateString('en-NG', {
    day: 'numeric', month: 'short',
  });
};

export const formatDeliveryEarning = (deliveryFee) => {
  // Rider earns 60% of the delivery fee
  return Math.floor(deliveryFee * 0.6);
};

export const getGreeting = () => {
  const hour = new Date().getHours();
  if (hour < 12) return 'Good morning';
  if (hour < 17) return 'Good afternoon';
  return 'Good evening';
};
```

## ERROR HANDLING PATTERN

Apply consistently across all API calls:

```js
try {
  setLoading(true);
  await store.action();
} catch (error) {
  const message = error.response?.data?.message || 'Something went wrong';

  // Race condition — delivery already taken
  if (message.includes('already been taken')) {
    showToast({ type: 'warning', message: 'This delivery was just taken by another rider' });
    store.removeAvailableDelivery(deliveryId);
    return;
  }

  showToast({ type: 'error', message });
} finally {
  setLoading(false);
}
```

## BUSINESS RULES TO ENFORCE IN UI

1. **Rider must be approved** — if `rider.isApproved === false`, show PendingApprovalScreen. Block access to deliveries.

2. **One active delivery at a time** — if rider already has an active delivery (`ASSIGNED` or `PICKED_UP`), hide the "Accept" button on all other delivery cards. Show "Complete your current delivery first."

3. **Must be online to see deliveries** — if `isAvailable === false`, show "Go online to see available deliveries" on the Deliveries tab.

4. **Location required** — before going online or accepting a delivery, check and request location permission. If denied, show an explanation and redirect to settings.

5. **Delivery status is one-directional** — only show `PICKED_UP` button when status is `ASSIGNED`. Only show `DELIVERED` when status is `PICKED_UP`. Never allow going backwards.

6. **Race condition handling** — when two riders try to accept the same delivery, the API returns an error. Handle gracefully: remove the card from the list and show "Already taken by another rider."

7. **Location tracking only during active delivery** — start `watchPositionAsync` when delivery is accepted, stop it when delivered or on screen unmount.

8. **Earnings calculation** — rider earns 60% of the delivery fee. Show this on the delivery card and after completion.
